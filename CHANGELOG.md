# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2023-10-27

### Added
- Initial release of Intel HEX Merger Tool.
- Support for standard Intel HEX format parsing.
- Support for Extended Linear Address (Type 04) for 32-bit addressing.
- File merging logic with overlap detection (File B overwrites File A).
- Drag and drop UI for file uploading.
- Real-time display of address ranges and data sizes.
- Automatic checksum calculation for generated HEX files.
- Multi-language support setup (English UI).
